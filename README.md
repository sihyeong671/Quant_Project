## 함수 && 클래스 명 작성 규칙
1. 첫글자는 무조건 대문자
2. 띄어쓰기 =  대문자 + 언더바(_)
3. 함수는 웬만하면 동사로 시작 > 동사+목적어(명사)
> `ex) Func_Test_Are_You_Understand`

## 변수명 작성 규칙
1. 모두 소문자로
2. 띄어쓰기 = 언더바(_) 
> `ex) this_is_variables`
